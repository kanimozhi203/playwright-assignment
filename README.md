# Playwright Automation Project

 Project Overview
This project is an automation testing framework built using Playwright with TypeScript.

It tests the SauceDemo website for login, product, and cart functionalities.

 Tools Used
- Playwright
- TypeScript
- JSON (for test data)
- Page Object Model (POM)

Project Structure
- pages/ → Page Object classes
- tests/ → Test scripts
- test-data/ → External test data (JSON)

Features Covered
- Positive test cases
- Negative test cases
- Validation scenarios
- Data-driven testing
- Cross-browser testing
